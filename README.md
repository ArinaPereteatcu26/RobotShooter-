# First-Person Shooting Game
## Overview
This project is a first-person shooting game deveoped in Unity using C#. It focuses on core game play mechanics such as shooting, enemy AI, player interaction.
The game can be accessed and played following the link: https://play.unity.com/en/games/d77ff91f-9969-4069-be16-183701cf800e/deadsight

## Features
- First-person player controller
- Shooting system using Raycasting
- Enemy AI powered by NavMesh Agents
- Camera transitions and zoom functionality
- Weapon Pickup System
- Ammo Pickup System
- Scriptable Objects
- Probuilder
- Weapon animations

## Technologies Used
- Unity Engine
- C# Programming

## Controls
| Action      | Key / Input        |
| ----------- | ------------------ |
| Move Player | `W`, `A`, `S`, `D` |
| Shoot       | Right Mouse Click  |
| Toggle Zoom | Right Mouse Click  |
| Run         | Hold Shift         |
